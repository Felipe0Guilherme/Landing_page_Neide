# Landing Page — Dra. Dalvoneide da Silva 🦷

> Landing page profissional desenvolvida para a **Dra. Dalvoneide da Silva**, Ortodontista em Vila Matilde, São Paulo. Focada em captação de pacientes, agendamento via WhatsApp e apresentação dos serviços de ortodontia.

---

## 🦷 Sobre o Projeto

Site de uma página completo para consultório odontológico, com design moderno, animações suaves e múltiplas seções de conversão. O alto índice de JavaScript (27.2%) reflete as funcionalidades interativas: slider de antes/depois arrastável, carrossel de depoimentos, animações de scroll e integração direta com WhatsApp.

---

## 🗂️ Estrutura do Projeto

```
Landing_page_Neide/
├── index.html    # Estrutura completa da landing page (357 linhas)
├── style.css     # Estilos, layout responsivo e animações
└── script.js     # Interatividade: slider, carrossel, scroll animations
```

---

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica com 7 seções distintas (36.3%)
- **CSS3** — Layout responsivo, gradientes, blobs animados, font Poppins via Google Fonts (36.5%)
- **JavaScript** — Slider antes/depois, carrossel de depoimentos, animações on-scroll, integração WhatsApp (27.2%)

---

## ✨ Seções e Funcionalidades

| Seção | Destaque |
|---|---|
| **Hero** | Headline impactante com blobs animados e CTA de agendamento |
| **Sobre** | Perfil da Dra. Dalvoneide com cards de diferenciais |
| **Serviços** | 4 serviços: Aparelho Fixo, Estético, Alinhadores Invisíveis e Manutenção |
| **Benefícios** | Grid de benefícios com emojis e copy persuasivo |
| **Antes e Depois** | **Slider interativo arrastável** — revela resultado antes/depois |
| **Depoimentos** | **Carrossel com dots** de 4 avaliações de pacientes com ⭐⭐⭐⭐⭐ |
| **CTA Final** | Seção de urgência para conversão com botão de WhatsApp |
| **Contato** | Cards com WhatsApp, telefone, endereço, horários + **mapa Google Maps embed** |
| **Botão Flutuante** | WhatsApp fixo na tela com mensagem pré-preenchida |
